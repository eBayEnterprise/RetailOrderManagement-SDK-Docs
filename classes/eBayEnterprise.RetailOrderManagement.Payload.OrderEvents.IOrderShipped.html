getCustomerId() : string

Assigned by the system that is taking the order. This can be either an id created by the customer, for example, when the customer created an account, or, if the customer did not want to create an account, the system creates an id, which is then used for guest checkouts. Therefore this field always has a value, even if the customer does not actively create an account.
